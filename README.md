# 자동차 경주 미션

### 🚗 프로젝트 소개

입력받은 자동차 이름과 라운드 횟수로 랜덤하게 자동차들을 전진하여 최종 우승자를 가리는 게임

### 🛠 구현 기능 목록

* 이름을 입력받는 기능
  * [예외] 공백, null, 쉼표만 입력으로 들어왔을 때 
  * [예외] 이름이 5자 초과일때

* 라운드 횟수를 입력받는 기능
  * [예외] 숫자로 표현되지 않는 모든 경우 (공백, null, 문자)
  * [예외] 0 이나 음수 입력일 경우

* 자동차를 전진 혹은 정지 시키는 기능

* 최고기록을 갱신하는 기능

* 우승자 목록을 반환하는 기능
